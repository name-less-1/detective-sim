# 🕵️ DetectiveAI — AI-Powered Detective Investigation Simulator

A turn-based detective mystery game where every case is uniquely generated by an AI. Interrogate suspects, collect evidence, and solve the crime before your turns run out.

**Live Demo:** [detective-sim.onrender.com](https://detective-sim.onrender.com)

---

## What It Does

Each time you start a game, the AI generates a completely original crime case — new suspects, new evidence, new guilty party. No two games are the same.

You have a limited number of turns to:
- 🔍 Inspect the crime scene
- 📹 Review CCTV footage
- 📱 Analyze phone records
- 🗣 Interrogate suspects

When your turns run out (or when you're ready), name your suspect. The game reveals the truth, scores your performance, and grades you from S to D.

## How It Works

```
Player starts game
      ↓
Groq API generates unique case JSON
(suspects, clues, motive, guilty party)
      ↓
Player investigates using actions
(each action reveals a clue)
      ↓
Player makes accusation
      ↓
Verdict engine scores performance
(correct accusation + evidence found + efficiency)
```

---

## Scoring

| Component | Max Points |
|-----------|-----------|
| Correct accusation | 40 |
| Key evidence collected | 40 |
| Efficiency bonus | 20 |
| **Total** | **100** |

Grades: **S** (90+) · **A** (75+) · **B** (55+) · **C** (35+) · **D** (below 35)

## Running Locally

**Prerequisites:** Node.js 18+, a Groq API key (free at [console.groq.com](https://console.groq.com))

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/detective-sim.git
cd detective-sim

# Install dependencies
npm install

# Set up environment
cp .env.example .env
# Add your GROQ_API_KEY to .env

# Start the app
npm run dev
```

Open `http://localhost:3000`
